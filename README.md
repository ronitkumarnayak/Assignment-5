# Assignment-5
1.Dictinory 
It is the simple python program to store information of students like ther name and marks.
When we enter name of the student it retrieves the student marks.
if entered name does not exist in the dictioary ,the program display student not found

How it Works
1. A dictionary is created with student names as keys and marks as values.
2. The user is allowed to enter a student's name.
3. The program checks whether the name exists in the dictionary.
4. If found, the student's marks are displayed.
5. If not found, "Student not found." is displayed.
